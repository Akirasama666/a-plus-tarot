# A+ Tarot — Site completo

Site landing page para Apóliya / A+ Tarot.

## Como editar o WhatsApp

Abra o arquivo `index.html` e procure por:

```txt
5500000000000
```

Troque pelo número dela no formato:

```txt
55 + DDD + número
```

Exemplo fictício:

```txt
5561999999999
```

## Como editar Instagram

Procure por:

```txt
https://instagram.com/
```

E troque pelo link do Instagram da marca.

## Como ver no PC

1. Baixe e extraia o ZIP.
2. Abra a pasta.
3. Clique duas vezes em `index.html`.

## Como publicar

Opção simples:
- Suba os arquivos no GitHub.
- Ative GitHub Pages.

Opção profissional:
- Suba no GitHub.
- Conecte o repositório na Vercel.
